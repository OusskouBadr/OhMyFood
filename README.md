# Ohmyfood

Projet réalisé pour apprendre et pratiquer le développement d’une application web avec Next.js dans le cadre de la formation OpenClassrooms.

## Description

Ohmyfood est une application web permettant de découvrir des restaurants et leurs menus.

L’objectif du projet était de travailler la création d’une interface responsive, la structuration d’un projet Next.js, l’utilisation de composants React et l’intégration d’une maquette.

## Fonctionnalités

* Affichage des restaurants
* Pages dédiées aux menus des restaurants
* Navigation entre les pages
* Interface responsive desktop, tablette et mobile
* Composants réutilisables
* Intégration fidèle à la maquette fournie

## Technologies utilisées

* Next.js
* React
* JavaScript
* CSS Modules
* Git
* GitHub

## Installation

Cloner le projet :

```bash
git clone https://github.com/OusskouBadr/ohmyfood.git
```

Installer les dépendances :

```bash
npm install
```

Lancer le serveur de développement :

```bash
npm run dev
```

Le projet est ensuite disponible à l’adresse :

```bash
http://localhost:3000
```

## Scripts disponibles

Lancer le projet en développement :

```bash
npm run dev
```

Créer une version de production :

```bash
npm run build
```
